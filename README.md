Project: Trader Behavior vs Market Sentiment

Objective:
Analyze how Fear/Greed sentiment affects trader performance.

Steps:
1. Data Cleaning
2. Data Merging
3. Analysis (Profit, Activity, Trade Size, Win Rate)
4. Automated Insight Generation

How to Run:
python analyze.py --file historical_data.csv

Output:
- insights.json (structured insights)
- Visualizations

Key Insights:
- Traders perform better during Greed periods
- Higher trading activity during Fear
- Larger trades during Greed
